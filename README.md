# HR Recruitment Analytics – End-to-End Hiring Project

## Project Overview
This project simulates an end-to-end recruitment process using realistic ATS-style data.
It analyses recruitment funnel performance, hiring source effectiveness, cost per hire,
and time to hire using Excel-based analytics.

## Dataset Details
- Total Candidates: 1,200
- Hiring Period: Aug 2024 – Jul 2025
- Roles Covered: HR Executive, Sales Executive, Data Analyst, Software Engineer
- Hiring Sources: Job Portal, Employee Referral, LinkedIn, Consultancy, Career Page

## Key Analysis Performed
- Recruitment Funnel Analysis (Applied → Joined)
- Hiring Source Effectiveness
- Time-to-Hire Analysis
- Cost-per-Hire Analysis
- Executive Dashboard for HR decision-making

## Tools Used
- Microsoft Excel
- Excel Formulas (IF, COUNTIF, AVERAGEIFS, VLOOKUP)
- Dashboard Charts

## Key Insights
- Employee referrals provide the lowest cost per hire and fastest hiring.
- Consultancy hiring is expensive but useful for niche roles.
- Major candidate drop-offs occur between interview and offer stages.

## Role Relevance
This project is designed for HR Executive / Talent Acquisition roles focusing on
end-to-end recruitment and data-driven hiring decisions.
